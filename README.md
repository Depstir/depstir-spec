# Depstir Protocol Specification

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](LICENSE.md)
[![Spec Status](https://img.shields.io/badge/Status-Draft-yellow.svg)](SPECIFICATION.md)

Depstir is a decentralized social protocol that enables wallet-based identities to interact through a distributed relay network while maintaining cryptographic proof of ownership over content and relationships.
